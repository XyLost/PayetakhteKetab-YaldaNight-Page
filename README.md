# 🍉🌙 Yalda Night Landing Page

A modern and responsive Yalda Night landing page designed for promotional campaigns, seasonal discounts, and festive product showcases.

Built with HTML, CSS, JavaScript, SCSS, and Swiper.js, featuring a Persian-inspired design, interactive product sliders, and a smooth user experience.

## ✨ Features

- 🍉 Yalda Night themed design
- 🎨 Modern and clean UI
- 📱 Fully responsive layout
- 🎠 Interactive product sliders powered by Swiper.js
- ⚡ Lightweight and fast
- 🌙 Persian typography and styling
- 🛍️ Product showcase sections
- 🚀 Easy to customize and deploy

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- SCSS
- Swiper.js

## 💡 About Yalda Night

Yalda Night (Shab-e Yalda) is one of the oldest Persian celebrations, marking the longest night of the year. Families gather together, enjoy traditional foods, read poetry, and celebrate warmth and togetherness.
